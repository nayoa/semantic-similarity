# Semantic Similarity

Programme that prints the similarity between objects.

- [Python](https://www.python.org/downloads/)
  - Version >= 3.8
- [pip](https://pip.pypa.io/en/stable/cli/pip_install/)
  - Version >= 22.3.1

## Build & Run
```bash
pip3 install -r requirements.txt
python3 semantic.py
```
